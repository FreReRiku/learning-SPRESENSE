# learning-SPRESENSE

## このリポジトリについて

思いついた案をそのまま書いてみたり、

調べて出てきたスケッチを整形してみたり、

既存のスケッチを混ぜ合わせたものをひたすら載せていきます。

練習用のサンプルスケッチを改造して載せるかもしれません。

実行できないスケッチだらけになるかもしれませんが、アイデアの掃き溜めということでご了承ください。

## 実行環境

SPRESENSE側

- 使用機器: Spresense メインボード + Spresense 拡張ボード
- 製作環境: Arduino IDE
- シリアル転送レート: 115200

PC側

- OS: macOS Sonoma 14.5
